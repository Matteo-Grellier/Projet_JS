# SuperHeroProject

## Introdution_au_projet

Ce projet consiste à recuperer des données d'un fichier json et d'ensuite pouvoir :

- Les afficher dans un tableau 
- Trier ces données par colonnes
- Effectuer une recherche pour un element en particulier dans le tableau
- Afficher un certain nombre d'elements par pages
- Afficher la page suivant et la page precedente

## Comment_l'utiliser

Pour utiliser ce projet, vous devez simplement lancer le fichier ``.html`` dans le dossier projet.
